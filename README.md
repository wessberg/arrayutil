## Description

A collection of helpers functions for working with arrays.

## Installation

`npm install @wessberg/arrayutil`

## Usage

```typescript
import {flat} from "@wessberg/arrayutil";

// Flat is well-typed.
flat([[[1]], [[2]], [[3]]]); // Returns [1, 2, 3]
```
